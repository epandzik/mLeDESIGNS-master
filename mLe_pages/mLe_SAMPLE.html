I coded this site, how's that for a sample?

I'd taken a general Web Development certificate at Johnson County Community College here in the Kansas City Metro. While working on learning general coding, such as HTML/CSS, JavaScript, jQuery, pHp and mySQL, I was supposed to set up a portfolio web site. It seemed appropriate that I would recode my portfolio site. I'd set up a WordPress page previous to this, using their templates and tools. Problem is that the tools I wanted weren't available there. It's useful, don't get me wrong. WordPress is great for the plug and play. But not so much for showing off the coding skills that I'm developing.
